Logo System

The Luis Gilberto logo system is built for clarity, adaptability, and emotional resonance. It comprises three essential components: Primary Logo, Logomark, and Dotcom logo; each designed to serve distinct roles across platforms and contexts. Unified by a signature symbol that fuses "L" and "G," this system balances strategic messaging with visual sophistication, ensuring brand consistency from mobile screens to marquee placements.

Primary Logo

The Primary Logo leads with a custom symbol that fuses the letters "L" and "G," forming a vertical anchor that supports the stacked "Luis Gilberto." To the right, a smaller coral phrase: "Marketing & Creative" adds clarity by signaling the brand's function and focus.

Logomark

The Logomark is a compact version tailored for mobile and space-constrained environments. It features four color variations: black and coral, each on light and dark backgrounds, for adaptable use across digital platforms. Recommended for dark backgrounds to maintain contrast and legibility.

Dotcom logo

Horizontal Layout

Stacked Layout

The Dotcom logo features the full domain name, with both horizontal and stacked layouts available. These versions prioritize clarity and recognizability in digital contexts, connecting the personal brand to its web identity.
